## ▶️ How to Run the Project

1. Open the project folder in **VS Code** or any code editor.

2. Open the **terminal** inside the project directory.

3. Install the required dependencies:

```
npm install
```

This command will install all necessary packages listed in `package.json` and create the `node_modules` folder.

4. Start the development server:

```
npm run dev
```

5. After running the command, the terminal will display a local URL (usually `http://localhost:5173`).
   Open that link in your browser to view the dashboard.
